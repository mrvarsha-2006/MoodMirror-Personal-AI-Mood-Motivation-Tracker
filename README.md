# 🌈 MoodMirror - Personal AI Mood & Motivation Tracker

MoodMirror is a personal mood tracking and motivation system designed to help build self-awareness, discipline, and emotional balance.  
It is created to support daily reflection, consistency tracking, and personalized motivational insights for self-growth — especially as a Biomedical Engineering student.

---

## ✨ Features

- 😊 Mood tracking (Happy, Okay, Sad, Tired, Stressed)
- 🧠 Personalized human-like motivational messages
- 📊 Mood history tracking & pattern detection
- 🔥 Daily streak tracking system
- 📝 Optional thought/journal entry support
- 💾 Local data storage using CSV
- 🌙 Dark-themed aesthetic UI
- 🚀 Auto-start support on system boot
- 💬 Mood-aware “AI-style” personalized quotes

---

## 🧠 How It Works

MoodMirror analyzes your emotional patterns and:

- Tracks daily moods and thoughts
- Identifies emotional trends (stress, fatigue, positivity)
- Generates personalized motivational messages
- Encourages discipline, balance, and consistency

---

## 🏗️ Tech Stack
